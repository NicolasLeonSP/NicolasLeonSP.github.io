Error Tipo Reserva

El Tipo de Reserva que se pedia y el que tenia yo en la base de datos eran distintos. La solucion era simple, cambiar el tipo de reserva al correcto.

Error Tipo IVA

El Tipo de IVA que yo tenia no coincidia con el que necesitaria usar para Java, como antes, la solucion era simple. Cambiar el campo de la base de datos a INT, y poner numeros en vez de letras.

Error toString null

Este error consistio en mal manejo de strings y excepciones. Basicamente, trataba de convertir un objeto que no existia en String. Esto se fue repitiendo alrededor del proyecto, pero la captura que pondre sera el que mas problemas dio por ello.

Error menuBarControler

Este error consistio en iniciar el objeto mas que el controlador. Aunque parezca un error tonto, me costo 1 hora descubrirlo, por lo tanto, se merece estar aqui.
